# NSE Market Intelligence Suite

## Objective
A high-performance financial data engineering pipeline designed for the analysis of institutional activity and volatility metrics within the National Stock Exchange (NSE) of India. This project serves as a quantitative evidence base for automated trade analysis and predictive modeling.

## The Stack
- **Languages:** Python 3.10
- **Libraries:** Pandas (Data Manipulation), NumPy (Numerical Analysis), Scikit-Learn (Predictive Modeling), Requests/YFinance (API/Data Ingestion)
- **Environment Management:** Conda (Isolated Sandbox)
- **Infrastructure:** Git/GitHub

## Key Projects
1. **Institutional Flow Extraction:** Automated scraping and cleaning of daily FII/DII institutional reports to monitor smart-money sentiment.
2. **Volatility Dashboard:** SQL-backed tracking of Nifty options volatility and Open Interest (OI) analysis.
3. **Short-Covering Breakout Model:** Machine learning application using Scikit-Learn to identify high-probability price breakouts based on volume and price action.
